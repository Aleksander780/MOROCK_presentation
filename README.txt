## Управление

Игра поддерживает управление с клавиатуры и мыши. Ниже приведен полный список клавиш:

| Клавиша                | Действие                                             |
|------------------------|------------------------------------------------------|
| **W, S, A, D**         | Ходьба / Перемещение                                 |
| **L. SHIFT**           | Бег (удерживать)                                     |
| **L. CTRL**            | Присесть                                             |
| **F**                  | Взаимодействие с объектами                           |
| **ЛКМ**                | Взмахнуть / Зарядить / Использовать предмет          |
| **ПКМ**                | Рисование рун (активация магии)                      |
| **СКМ (Колесо)**       | Выбросить предмет из рук                             |
| **TAB**                | Открыть инвентарь                                    |
| **ESC**                | Пауза / Выход в меню                                 |
| **SPACE (удерживать)** | Прыжок. В воздухе позволяет использовать **паркур**. |

## Настройки

> **Внимание:** В текущей версии меню настроек работает ограниченно. Через внутриигровое меню можно изменить только параметры графики.

Чтобы изменить расширенные настройки или отредактировать файл конфигурации вручную, перейдите по следующему пути:
C:\Users{Имя_пользователя}\AppData\LocalLow\DESIRION STUDIO\MOROCK


Замените `{Имя_пользователя}` на имя вашей учетной записи в Windows.

## Тестовая зона

Для быстрого доступа к игровым механикам и окружению предусмотрена "Тестовая зона".

- **Как попасть:** В главном меню игры нажмите кнопку **"Продолжить"**.

## Боевая система

В игре реализована динамическая боевая система, основанная на движении мыши.

1.  **Заряд:** Зажмите **левую кнопку мыши (ЛКМ)**.
2.  **Направление:** Не отпуская кнопку, двигайте мышью в сторону предполагаемого удара.
3.  **Удар:** Отпустите **ЛКМ**, чтобы нанести удар в выбранном направлении.

## Магия

Магия активируется путем рисования рун прямо в воздухе.

1.  **Активация:** Зажмите **правую кнопку мыши (ПКМ)**.
2.  **Рисование:** Удерживая **ПКМ**, рисуйте в воздухе необходимый символ.
    *(Примечание: Список доступных рун прилагается к игре в docx-файле).*

## Важно

- **Сетевой доступ:** Текущая сборка игры собрана в **отладочном режиме**, поэтому игра может пытаться получить доступ к сети. Это нормально для данной версии.
- **Сохранения:** Файлы сохранений и конфигурации хранятся по пути, указанному в разделе [Настройки](#настройки).
